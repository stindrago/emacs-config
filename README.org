* Installation

In order to have Doom load the config, we need to tangle =config.org= and
generate the =*.el= files.

#+begin_src shell :eval query :tangle no
git clone https://gitlab.com/stindrago/emacs-config ~/.config/doom
emacs --batch --eval "(progn (require 'org) (setq org-confirm-babel-evaluate nil) (org-babel-tangle-file \"~/.config/doom/config.org\"))"
git clone --depth 1 https://github.com/hlissner/doom-emacs ~/.emacs.d
~/.emacs.d/bin/doom install; ~/.config/doom/setup.sh
#+end_src
